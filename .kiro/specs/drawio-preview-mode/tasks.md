- [x] 2. アセット読み込み基盤の強化
- [x] 2.1 アセットローダの除外マニフェスト対応
  - アセットローダに除外述語オプションを追加し、列挙段階で対象外ファイルをスキップする
  - design.md の EDITOR_ASSET_EXCLUDES 初期マニフェストを 1 定数として定義する
  - ユニットテストで除外対象がバンドルに含まれないことが検証されている
  - _Requirements: 5.2_
- [x] 2.2 セッションアセットキャッシュ
  - エディタバンドルと viewer スクリプトを遅延ロード + メモ化するキャッシュを実装する (single-flight)
  - ロード失敗時はメモを破棄し次回再試行できる。dispose で全メモ解放、dispose 後の呼び出しは再ロードで応える
  - ディスクロードの実行回数をデバッグログで観測可能にする (E2E 5.2 の計装アサーションの根拠となる)
  - ユニットテストで並行 2 呼び出し時にローダが 1 回しか呼ばれないこと、失敗→リトライ、dispose 後再ロードが検証されている
  - _Requirements: 5.2, 5.3, 5.4_
- [x] 2.3 bridge のアセット注入とオブジェクト送信化
  - drawio-bridge がアセット provider を注入で受け取り、自前ローダはフォールバックに退避する
  - 親→iframe の configure / script メッセージをオブジェクトのまま postMessage し、frame-messenger が文字列 JSON とオブジェクトの両方を受理する
  - 既存の編集モード挙動 (保存・autosave・外部変更) が変わらないこと (既存ユニットテスト green + object 受理テスト追加)
  - _Requirements: 3.5, 5.2_
  - _Depends: 2.2_
  - _Boundary: drawio-bridge, frame-messenger_

- [x] 3. プレビュー表示経路
- [x] 3.1 (P) preview iframe 内 init スクリプトとビルド
  - viewer iframe 内で GraphViewer を起動する init スクリプトを実装する (render 受信 → 描画、ready / error を親へ通知。frame-globals を流用し DRAWIO_BASE_URL を設定)
  - GraphViewer は toolbar (pages / zoom / layers) 付きで生成し、ズーム・パン・ページ切替・初期フィットを組み込み機能で提供する
  - IIFE ビルド設定を追加し、build スクリプトで preview-init.js が dist に出力される
  - _Requirements: 1.3, 2.1, 2.2, 2.3, 2.4_
  - _Boundary: preview-init_
- [x] 3.2 (P) PreviewBridge の実装
  - viewer iframe の親側ライフサイクル (idle → loading → ready | error → disposed) を実装する
  - bootstrap iframe 生成 → preview-init / viewer スクリプト注入 → render 送信の順で描画し、タイムアウト超過は error コールバックへ
  - ファイル書き込みに至る API を持たない (読み取り専用の構造的保証)
  - frame メッセージングをモックしたユニットテストで mount → ready 到達、error / timeout 分岐、dispose 時の iframe 破棄が検証されている (3.1 の実体には依存しない)
  - _Requirements: 1.3, 2.5, 5.1_
  - _Depends: 2.2_
  - _Boundary: PreviewBridge_
- [x] 3.3 (P) 画像プレビュー UI とエラーパネル
  - 内包画像 (svg / png) を vault リソース URL 直接指定で表示し、ホイール / ピンチ / ドラッグ / ズームボタンで拡大縮小・パンできる React コンポーネントを実装する (ファイル内容の再読込や Blob URL 管理は行わない)
  - 初期表示は図全体がビューに収まるフィット表示
  - 描画失敗時のエラーパネル (メッセージ + 「エディタで開く」導線) を実装する
  - _Requirements: 1.2, 1.5, 2.1, 2.2, 2.3, 2.5_
  - _Depends: 1.3_
  - _Boundary: views/preview_
